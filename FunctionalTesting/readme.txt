# 🧪 Edupulse QA Automation (Selenium + PyTest)

This project automates the login and test-taking flow for [Edupulse QA](https://edupulse-qa.onrender.com), including:
- Logging in
- Starting or resuming a Practice Test
- Submitting the test (with browser alerts handled)
- Logging out

Supports both **headed** and **headless** execution.

---

## 🛠️ Requirements

- Python ≥ 3.8  
- Google Chrome browser installed  
- pip (Python package installer)

---

## 📦 Dependencies

Install all required packages using the command below.
